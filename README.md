# Ather_Round-3_Task
Built a console Based 2048 Game in python.


# How to Play!
Launch the 2048-Game.py file in your console using the given command.
python 2048_Game.py
After launching follow the game instructions:
Enter 1 for Left.
Enter 2 for Bottom.
Enter 3 for rigjt.
Enter 4 for Top.
Enter 5 for Quit.

# Requirements
Python 3.7+
